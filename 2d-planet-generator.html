2D Planet Generator Features

2D Planet Generator uses layered perlin noise to create a believable terrain. Octaves, frequency, lacunarity, amplitude and persistence of the perlin noise can be adjusted to create the right planet. Masking of noise by another noise is possible with this asset. Masking allows more control over terrain generation. Water can be added to create oceans and lakes. A generated planet can be exported as PNG.
